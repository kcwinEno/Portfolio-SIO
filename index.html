BTS SIO & Option SLAM

Le BTS SIO (Services Informatiques aux Organisations) est une formation post-bac de deux ans en France, destinée à former des techniciens supérieurs en informatique. Le BTS SIO a pour objectif de doter les étudiants de compétences essentielles dans la gestion, la mise en place et le maintien des systèmes d’information dans les entreprises. L'option SLAM (Solutions Logicielles et Applications Métiers) est plus orientée vers le développement. Elle forme des professionnels capables de concevoir et de développer des applications (sites web, applications mobiles, logiciels) et de rédiger de la documentation technique associée aux projets développés.

    Analyse : Application Bureau Ligue1 (C#)

    Objectif

    Développer un logiciel de bureau permettant à un administrateur de gérer le calendrier, de saisir les scores des matchs et de générer automatiquement le classement du championnat.

    Analyse Technique

    Ce projet m'a permis de mettre en pratique les concepts de la Programmation Orientée Objet (POO) en C#

    L'interface utilisateur a été conçue pour être intuitive, avec une gestion d'événements permettant de mettre à jour le classement en temps réel dès la validation d'un score. Les données sont persistées via une base de données locale SQL Server.

    Amélioration possible : Intégrer un système d'import/export de données au format JSON ou XML pour faciliter les échanges avec d'autres systèmes ou le site web PHP.

    ×

    Analyse : Site Web Ligue1 (PHP)

    Objectif

    Développer une plateforme web dynamique permettant aux utilisateurs de consulter les résultats, les classements et les statistiques du championnat en temps réel.

    Analyse Technique

    Le site repose sur une architecture LAMP (Linux, Apache, MySQL, PHP).

    La logique métier est séparée de l'affichage pour garantir un code plus propre. J'ai également mis en place une gestion de l'affichage dynamique des données en fonction des résultats stockés en base.

    Amélioration possible : Intégrer une architecture de type MVC (Modèle-Vue-Contrôleur) pour mieux structurer le projet et faciliter l'évolution vers un framework comme Laravel par la suite.

    ×

    Analyse : Application Bureau Ada (C# / Entity Framework)

    Objectif

    Concevoir une application dédiée à la gestion interne des réservations, du parc automobile et de la facturation pour les agents en agence.

    Analyse Technique

    L'utilisation de l'ORM Entity Framework a permis de faire le pont entre le modèle objet en C# et la base de données SQL Server. Grâce aux requêtes LINQ, j'ai pu manipuler les données de manière sécurisée et typée, évitant ainsi les injections SQL.

    L'architecture respecte une séparation des couches, facilitant la maintenance corrective de l'application.

    Amélioration possible : Implémenter le pattern Unit of Work ou Repository pour isoler encore davantage la logique d'accès aux données et faciliter les tests unitaires.
